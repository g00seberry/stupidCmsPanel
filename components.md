- на странице /content-types/article/ должна появиться кнопка "настроить форму", подобно "Настроить Blueprints"
- на ней должна загрузиться схема blueprint выбранного для этого типа поста
- для каждого узла схемы должно быть можно выбрать компонент и задать его настройки
- тип компонента, который можно выбрать для того или иного узла определяется при помощи src\components\schemaForm\componentDefs\getAllowedComponents.ts
- форма настроек компонента должна генерироваться из метаданных выбранного типа src\components\schemaForm\componentDefs\ZComponent.ts
- после внесения всех необходимых данных по нажатию на кнопку "сохранить" данные сохраняются на бэкенд.
