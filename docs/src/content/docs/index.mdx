## Быстрый старт

- Начинай с раздела [`00-overview`](./00-overview/index.md).
- Для «карточек» модулей смотри каталог [`10-modules`](./10-modules/).
- Компоненты описываются в [`30-components`](./30-components/).
- Публичный API собирается в [`refs`](./refs/).

## Регламент

1. Обеспечивай однострочный `summary` у каждого экспортируемого символа.
2. После изменений выполняй `npm run docs:gen` и добавляй артефакты в коммит.
3. Перед пушем проходит `npm run docs:check` (pre-commit hook).
