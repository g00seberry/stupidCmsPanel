## Навигация

- `00-overview/` — ручные вступительные материалы.
- `10-modules/` — автоген «карточек» модулей.
- `20-apis/` — спецификации OpenAPI (если есть).
- `30-components/` — автоген по React-компонентам.
- `refs/` — справочник публичного API.

## Как обновлять

Запускай `npm run docs:gen` после добавления/изменения публичных экспортов и коммить результаты вместе с исходниками.
