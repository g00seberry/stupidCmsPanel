# Гайды

Этот раздел содержит ручную документацию: процессы, архитектурные решения (ADR), инструкции (HowTo) и другие материалы, которые не генерируются автоматически.

## Что здесь хранить

- Архитектурные решения (ADR)
- Процессы разработки
- Инструкции по настройке и развёртыванию
- Best practices
- Примеры использования

## Структура

Создавайте отдельные файлы для каждой темы, например:
- `architecture.md` — архитектура проекта
- `deployment.md` — процесс развёртывания
- `development.md` — процесс разработки
