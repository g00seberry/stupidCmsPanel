# Документация проекта

Добро пожаловать в документацию CMS Panel.

## Разделы

- **Гайды** — процесс, архитектура, ADR, HowTo, [руководство по документированию](/guides/documentation-guide)
- **API** — автогенерация из TypeScript
- **Компоненты** — автогенерация из React props
- **Страницы** — автогенерация из файлов `src/pages`

Внутри `guides/` храните обзорные материалы и решения продуктовых кейсов — это единственный «ручной» раздел.

## Правила документирования

Все правила по документированию кода находятся в `.cursor/rules/documentation.mdc` и применяются автоматически при работе в Cursor.

**Основные принципы:**
- Все публичные API должны иметь JSDoc комментарии
- Документация обновляется автоматически при коммите
- Примеры использования обязательны для сложных функций
- Документация всегда должна быть актуальной

## Запуск локального сервера документации

```bash
npm run docs:dev
```
